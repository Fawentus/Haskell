# Haskell

### Полезные функции:

Подгрузить файл, где будут описаны функции:
```
:load "<name_file>"
```

Обновить подгруженные файлы:
```
:reload
```

Изменить вид "приветственной" строки:
```
:set prompt <name>
```

Как комментарии отставлять:

```
this code -- this comment
```
